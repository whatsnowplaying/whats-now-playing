# Lumia Stream

**What's Now Playing** supports [Lumia Stream](https://lumiastream.com/) via a dedicated plugin.
The plugin connects to WNP's local web server, exposes full track metadata as Lumia variables,
and fires a **Track Changed** alert on every track change.

Full installation instructions, variable reference, and troubleshooting are in the plugin
repository:

<https://github.com/whatsnowplaying/wnp-lumiaplugin>

## Requirements

* What's Now Playing with the **Web Server** output enabled
* Lumia Stream 9.0 or newer
