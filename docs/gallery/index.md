# Gallery

**What's Now Playing** supports a wide variety of outputs. Here are some examples of each in action.

* [Webserver Templates](webserver.md): HTML overlays displayed in OBS via Browser Source
* [WebGL Templates](webgl.md): GPU-accelerated animated overlays requiring WebGL support
* [Dynamic Backgrounds](dynamic-backgrounds.md): Full-screen WebGL backgrounds that adapt to cover art palette colors
* [OBS WebSocket](obswebsocket.md): Text output written directly to an OBS text source
* [Unicode Support](unicode.md): Non-ASCII characters in track metadata
