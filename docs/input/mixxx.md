# MIXXX

[MIXXX](https://mixxx.org/) is free, open-source DJ software. Which input source to use depends
on your build of MIXXX:

## Standard MIXXX

If you are using an official MIXXX release, use the **[Icecast](icecast.md)** input source.
MIXXX broadcasts to the Icecast server built into **What's Now Playing**, which reads the track
information directly from the stream.

See [Icecast → Settings For MIXXX](icecast.md#settings-for-mixxx) for setup instructions.

## MIXXX with MPRIS2 Support (Linux)

Some Linux builds of MIXXX include MPRIS2 support. If your build supports MPRIS2, use the
**[MPRIS2](mpris2.md)** input source instead — it provides a more direct integration.

See [MPRIS2](mpris2.md) for setup instructions.
